## Comment créer votre CV ?

### Options gratuites :
1. **Canva** - https://www.canva.com/fr_fr/creer/cv/
   - Templates modernes
   - Facile à utiliser
   - Export PDF gratuit

2. **Figma** - https://www.figma.com/
   - Professionnel
   - Templates CV gratuits
   - Export PDF

3. **LaTeX (Overleaf)** - https://www.overleaf.com/
   - CV technique très propre
   - Template `moderncv`
   - Idéal pour développeurs

4. **Microsoft Word / Google Docs**
   - Templates intégrés
   - Export en PDF

### Conseils pour votre CV :
- ✅ 1 page maximum
- ✅ Photo professionnelle
- ✅ Coordonnées claires (email, GitHub, LinkedIn)
- ✅ Projets avec liens GitHub
- ✅ Compétences techniques
- ✅ Formations et certifications
- ✅ Expériences (stages, projets scolaires)
